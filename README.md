# 🎮 Unity Endless Runner (Учебный проект)

Данный репозиторий содержит учебный проект на Unity, реализующий механику **бесконечного раннера (Endless Runner)** с расширяемой архитектурой и набором игровых систем.

Проект ориентирован на демонстрацию практических подходов к разработке игр: оптимизация, архитектура, работа с данными и масштабируемость.

---

## 🚀 Основные возможности

### 🏃‍♂️ Core-геймплей

* Реализована механика **бесконечного раннера**
* Постоянное движение персонажа вперед
* Генерация уровня «на лету»

---

### ♻️ Система пула объектов

* Используется **Object Pooling** для сегментов уровня
* Повышает производительность за счёт переиспользования объектов
* Уменьшает количество аллокаций и GC

---

### ⚠️ Динамическая генерация препятствий

* Препятствия создаются во время игры
* Генерация адаптируется под текущие условия

---

### 🧠 Расширяемые паттерны препятствий

Поддерживается система шаблонов (паттернов), которые легко расширяются.

На данный момент реализованы:

* 🐍 **Змейка**
* ⬅️ **T-образный паттерн**
* 🎲 **Случайная генерация**

---

### ⚙️ Гибкая настройка уровня

* Используются **ScriptableObject** для конфигурации
* Можно настраивать:

  * скорость игры
  * частоту появления препятствий
  * сложность
* Удобно для балансировки без изменения кода

---

### 🛒 Магазин скинов

* Покупка скинов за внутриигровую валюту
* Хранение купленных предметов
* Выбор активного скина

---

### 🎨 Система скинов

* Все скины описаны через **ScriptableObject**
* Простое добавление новых скинов без изменения логики

---

### 🎯 Система достижений 

* Все достиежения описаны через **ScriptableObject**
* Простое добавление новых достижений
* В логику встраевается проверка условия достижения 
* Достижения сохраняются при получении 

---

### 💾 Сохранение данных

* Используется `DataController` (Singleton)
* Хранит:

  * прогресс игрока
  * купленные скины
  * выбранный скин
* Архитектура позволяет легко расширять функциональность

---

### 📢 Система рекламы (заглушка)

* Реализован Singleton-контроллер рекламы
* Содержит все необходимые методы:

  * показ баннеров
  * показ rewarded-рекламы
  * интерстициальная реклама
* **API не подключён (заглушка)** — легко интегрировать любую рекламную сеть

## 🧩 Архитектурные особенности

* Использование **Singleton-паттернов** для глобальных систем
* Разделение данных и логики через **ScriptableObject**
* Масштабируемая система генерации препятствий
* Оптимизация через Object Pooling
* Паттерн **Наблюдатель** для систем которым важны изменения конфигов уровня

---

## 🛠️ Технологии

* Unity (6.3.8f1)
* C#
* ScriptableObject
* Асинхронная загрузка сцен
* Паттерны проектирования:

  * Singleton
  * Object Pool
  * Observer
